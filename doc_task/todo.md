# 代辦清單 (Todo List)

這個文件介紹 `doc_task` 這個資料夾的使用與管理方式。

### `doc_task` 資料夾內容說明：
- `todo.md`: 負責記錄任務執行摘要。
- `task_template.md`: 作為新建立任務時的執行文件範本。
