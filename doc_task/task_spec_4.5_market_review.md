## 3. 內容結構藍圖 (Content Blueprint)

### A. 黃金標題 Strategy (H1)
- [ ] **公式**: 【2025最新】+ 數字 + 產品類別推薦 + 獨特賣點
- [ ] **範例**: 「【2025最詳細】8款最有特色的魚油推薦，每款適合的需求都不同！」

### B. 信任聲明 (The Hook & Trust Signals)
- [ ] **痛點切入**: 質疑「你真的需要吃魚油嗎？」。
- [ ] **信任三支柱**:
    1. **如何評測**: 強調「自費購買」、「客觀中立」。
    2. **評測團隊**: 醫師/營養師背書。
    3. **評測標準**: 基於科學數據 (濃度、檢驗)。

### C. 基礎教育 (Educational H2)
- [ ] **功效 Hashtag**: #乾眼 #高血壓 #記憶力，讓讀者對號入座。
- [ ] **選購原則**: 設定標準 (濃度>80%, rTG型態)，為產品推薦鋪路。

### D. 產品推薦列表 (The Core H2)
- [ ] **總表**: 快速清單 (產品名/濃度/價格)。
- [ ] **單品評測 (重複 8 次)**:
    - **Header**: 品牌名 + 關鍵規格 (e.g., 1000mg/85%)。
    - **Specs Card**: 列表呈現含量、來源、檢驗狀況。
    - **Review Body (3段式)**:
        1. 品牌與規格分析。
        2. 原料與安全性查核 (檢驗報告)。
        3. **體驗與適口性** (關鍵): 吞嚥感受、腥味測試 (Experience)。
    - **Verdict**: 適合誰 (重視CP值 vs 討厭吞藥)。
    - **Pros/Cons**: 誠實揭露缺點。
    - **CTA**: 醒目購買按鈕。

### E. 評測實驗室 (Lab Test Visuals)
- [ ] **剪開膠囊**: 影片/GIF 展示色澤。
- [ ] **腥味盲測**: 雷達圖 (腥味、檸檬味)。
- [ ] **顆粒大小**: 硬幣對比圖。

### F. 總結與比較 (Conclusion)
- [ ] **懶人挑選法**: 「預算夠選 A，怕腥味選 B」。
- [ ] **比較表格**: 橫向滑動表格 (Responsive Table)。

## 4. 撰寫指南 (Writing Guidelines)
- [ ] **視覺化**: 關鍵數據用 **粗體** 或 顏色 標示。
- [ ] **短段落**: 手機閱讀友善，每段 < 4 行。
- [ ] **語氣**: 客觀分析 ("實測", "比較")，避免過度推銷。
- [ ] **SEO**: 佈局長尾關鍵字 (rTG, 高濃度, 兒童)。

## 5. 驗收標準
- [ ] 標題包含年份與差異化賣點。
- [ ] 每款產品皆有實拍圖 (非廠商宣傳照)。
- [ ] 規格表格式統一。
- [ ] 比較表格在手機版無跑版。
