## 附錄: 執行時的注意事項

### 給 AI 模型的執行提示

1. **一次只做一個 Step**: 請嚴格按照 Step 1.1 → 1.2 → 1.3... 的順序執行,不要跳躍或合併
2. **使用 `replace_file_content` 工具**: 每次修改時,明確指定 `StartLine` 與 `EndLine`,並提供完整的 `TargetContent` 與 `ReplacementContent`
3. **先 View 再 Edit**: 在執行每個 Step 之前,先用 `view_file` 確認插入點的上下文環境
4. **驗證步驟不能省**: 每完成一個 Step,必須再次 `view_file` 檢查修改後的結果
5. **遇到衝突立即停止**: 如果發現既有內容與計劃描述不符(如品牌卡片位置改變),請通知使用者,不要強行插入

### 常見錯誤預防

- **錯誤1**: 一次性修改過多行導致 `TargetContent` 無法精準匹配 → **解法**: 嚴格控制每次修改在50-150行內
- **錯誤2**: HTML 標籤未閉合導致後續版面崩潰 → **解法**: 使用程式碼編輯器的括號匹配功能,或在瀏覽器中即時預覽
- **錯誤3**: 數據不一致(如同一品牌的價格在不同段落有差異) → **解法**: 建立數據對照表,統一引用來源
- **錯誤4**: CSS class 拼寫錯誤導致樣式失效 → **解法**: 先用 `grep_search` 搜尋現有頁面的 class 用法,複製貼上確保一致
