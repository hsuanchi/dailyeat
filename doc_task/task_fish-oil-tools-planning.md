## 1. 任務目標 (Objective)

設計一系列**「SEO 流量磁鐵」型**的魚油相關網頁工具。目標不僅是實用，更要**最大化自然搜尋流量**，捕捉長尾關鍵字（如：攝取量、副作用）與**高價值的「品牌關鍵字」**（如：某某牌魚油濃度、某某牌CP值）。

透過讓使用者輸入數據或選擇情境，讓工具頁面成為使用者驗證產品好壞的權威入口，進而提升頁面停留時間與互動數據。

## 2. 實作策略 (Implementation Strategy)

### 2.1. SEO 核心策略

*   **品牌關鍵字攔截**: 設計工具讓使用者可以輸入「自己手上的品牌數據」，藉此在頁面內容中自然包含各家品牌的比較邏輯（無需直接提到品牌名即可攔截搜尋意圖，或在結果中動態生成比較文案）。
*   **長尾問題解答**: 將「孕婦怎麼吃」、「可以跟鈣一起吃嗎」等長尾問題轉化為互動選項。
*   **結構化資料**: 確保工具產出的結果能被 Google 理解（若技術允許）。

### 2.2. 工具規劃 (SEO 導向)

預計規劃以下 3 個核心工具，優先順序如下：

### 2.2. 工具規劃 (SEO 導向)

預計規劃以下 3 個核心工具，優先順序如下：

1.  **魚油「純度與價值」分析儀 (Purity & Value Analyzer)**
    *   *SEO 目標*: 鎖定「某品牌魚油濃度」、「魚油CP值計算」、「高濃度魚油標準」等關鍵字。
    *   *功能*: 使用者輸入手邊產品的「每顆重量」、「EPA/DHA 含量」與「價格」。
    *   *輸出*: 客觀分析「純度是否達高標」、「每一塊錢買到多少有效成分 (CP值)」。
    *   *價值*: 讓使用者透過數據「科學分析」產品價值，強化我們內容的專業與公正性。

2.  **個人化魚油挑選/檢測器 (Smart Selector)**
    *   *SEO 目標*: 鎖定「孕婦魚油推薦」、「考生魚油劑量」、「三高吃魚油」等族群+需求關鍵字。
    *   *功能*: 選擇身份（孕婦/銀髮/上班族）與需求（護眼/護心/考試）。
    *   *輸出*: 客製化的劑量建議、成分比例建議（EPA vs DHA）與貼心提醒（如：交互作用提示）。

3.  **魚油 vs 藻油 vs 磷蝦油 終極比較表 (Interactive Comparison)**
    *   *SEO 目標*: 鎖定「素食魚油」、「磷蝦油比較」等競品字詞。
    *   *功能*: 互動式開關，讓使用者勾選自己在意的點（如：素食、膽固醇、吸收率），動態顯示最適合的選擇。

### 2.3. 技術方案

*   **前端框架**: 純原生 JavaScript (Vanilla JS)。
*   **樣式隔離**: 使用 `Shadow DOM` (Mode: Closed) 封裝。
*   **動態 SEO**: 雖然工具在 Shadow DOM 內，但需確保*輸入前的引導文案*與*頁面靜態內容*包含豐富的關鍵字佈局。
